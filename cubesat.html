Weather Forecasting Based on CubeSat

Academic Project

Project Overview

Designed and developed a real-time weather forecasting CubeSat system using Arduino Uno. The CubeSat collects live environmental data such as temperature, humidity, rainfall, wind speed, and pressure, and transmits it to a ground station using LoRa communication.

The system includes a GSM module for sending SMS alerts to users in remote areas and a buzzer alert system for critical weather conditions. Powered by solar energy, the solution is sustainable, low-cost, and ideal for real-time monitoring without internet access.

Applications & Uses

  • Helps rural, hilly, and tribal regions receive weather alerts without internet
  • Supports agriculture planning and disaster preparedness
  • Useful for CubeSat, IoT, and environmental research studies
